Descripción del servicio

Daico es una aplicación web privada para organización del hogar: tareas, turnos y documentos. El acceso es restringido a cuentas autorizadas.

Uso aceptable

La aplicación es para uso personal. Al usarla, aceptás no intentar acceder a datos de otras personas ni interferir con el funcionamiento del servicio.

Cuentas

Iniciás sesión con tu cuenta de Google. Sos responsable de mantener la seguridad de tu cuenta. Podemos suspender el acceso si detectamos uso indebido.

Disponibilidad

No garantizamos disponibilidad ininterrumpida del servicio. Podemos modificar o discontinuar la aplicación en cualquier momento.

Limitación de responsabilidad

Daico se ofrece "tal cual". No nos hacemos responsables por pérdidas derivadas del uso de la aplicación.
